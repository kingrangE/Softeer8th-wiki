## 회고
### Keep

    - Action : 점심에 단백질/무기질 위주의 식사를 찾아보자.

### Problem
1. 간식도 당은 피하기
    - 당을 먹으면 졸리다. 오늘 괜히 안 졸린데 간식 먹고 약간 피곤해졌다.
    - Action : 최대한 간식을 피하거나, 당 없는 건강에 좋은 간식 찾기
2. 목표했던 시간을 모두 넘겨서 과제 완료
    - Action : 아래에 따라서 원인 찾기
        1. 과제마다 시간 기록
            - 시작 전 추정 시간 (E)
            - 실제 총 경과 시간 (A)
            - 실제로 진행하던 순수 작업 시간 (F)
        2. 문제 파악
            - A-F : 딴짓한 시간
                - 이게 많으면 딴짓을 어떻게 줄일지 고민
            - F-E : 능력이 부족한 차이
                - 내 실제로 능력이 어느정돈지 메타인지하기
3. 개념 정리 -> 미션 학습 순서로 인한 효율성 낮음
    - Action : 아래와 같이 공부하기
        1. 수업 종료 후, 가볍게 눈으로 복습
        2. 미션이 있다면
            - 미션 먼저 진행
            - 미션 하나가 PR까지 완료되면, 1개 당 1개씩 학습 내용 정리
        3. 미션이 모두 완료되었거나, 없다면
            - 수업 내용을 기반으로 주제 추출
            - 주제 별로 내용 정리 
                - 어차피 내가 이걸 엄청 자주 볼 것 같진 않다. 중요한 개념 위주로만 정리하자. (1h)
